# KOLLAB Job Board

KOLLAB is a premium, community-driven job board connecting creative professionals with opportunities in Design, Engineering, Marketing, and more.

## Features

-   **Job Search**: Filter by keywords and location (e.g., "Chicago", "Remote").
-   **Categories**: Highlighted filtering by job field.
-   **Job Details**: Comprehensive view with "About", "Requirements", and "Benefits".
-   **Authentication Flow**:
    -   **Login**: Professional split-layout design.
    -   **Sign Up**: Dedicated role selection for Recruiters and Job Seekers.
-   **Post a Job**: Simplified, user-friendly form for posting new opportunities.

## Technologies
-   HTML5
-   CSS3 (Custom properties, Flexbox, Grid)
-   Vanilla JavaScript
